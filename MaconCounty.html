Visit my Macon County Website Redesign

Macon County Website Redesign is a modernized version of a fictitious county government site I created using a Bootstrap template. The goal of this project was to improve the site for local residents by making it more user-friendly, responsive, and visually updated. I reorganized the layout to ensure faster loading times and easier navigation, especially on mobile devices. Key features I added include downloadable PDF forms, direct links to government resources, and a regularly updated events section so residents can stay informed. This redesign emphasizes both function and accessibility.
